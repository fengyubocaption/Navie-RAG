# DeepSeek-V3 技术架构白皮书（摘要版）

## 1. 模型概览
DeepSeek-V3 是一款强大的开源混合专家模型（MoE）。它旨在提供与顶级闭源模型相当的性能，同时大幅降低推理和训练成本。

## 2. 核心架构特性
* **MLA (Multi-head Latent Attention)**: 这是 DeepSeek-V3 的核心创新。通过对 KV Cache 进行低秩压缩，MLA 在推理时将显存占用降低了约 **93%**，极大地提升了长文本处理能力。
* **DeepSeekMoE**: 采用了细粒度专家路由策略。模型总参数量为 **671B**，但每次推理仅激活约 **37B** 参数。
* **无损 FP8 训练**: 它是首个在大规模训练中成功应用 FP8 混合精度方案的万亿级模型，显著提升了算力利用率。

## 3. 训练数据与规模
* **数据集大小**: 预训练阶段共使用了 **14.8 万亿 (14.8T)** 个 Token。
* **语料构成**: 包含大量的编程代码（20%）、数学公式（10%）以及多语言通用文本。

## 4. 性能指标 (Benchmark)
在最新的评测中，DeepSeek-V3 表现如下：
* **MMLU (综合知识)**: 88.5%
* **GSM8K (数学逻辑)**: 90.2%
* **HumanEval (代码能力)**: 82.6%

## 5. 常见问题解答 (FAQ)
**问：DeepSeek-V3 支持的最大上下文长度是多少？**
答：该模型原生支持 **128K** 的上下文窗口。

**问：该模型是否支持函数调用（Function Calling）？**
答：是的，模型针对 Tool Use 场景进行了专项强化训练。
